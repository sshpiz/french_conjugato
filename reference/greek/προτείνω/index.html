Usages & examples

προτείνω κάτι σε κάποιονΤης πρότεινα αυτό το βιβλίο.I recommended this book to her.
προτείνω να + υποτακτικήΣου προτείνω να φύγουμε νωρίς.I suggest we leave early.
προτείνω κάποιον για θέσηΤον πρότεινα για τη θέση του υπεύθυνου.I nominated him for the manager position.
όπως προτείνει...Ας το κάνουμε όπως προτείνει η Μαρία.Let's do it as Maria suggests.
