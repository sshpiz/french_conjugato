# French Core Patterns Style Guide

This layer is intentionally narrower than `verb_usages`.

## Goal

Show only the most useful learner-facing core constructions for a verb:
- direct object
- bare infinitive
- `à`
- `de`
- `que`
- `si`
- reflexive / impersonal patterns
- a few high-value constructional patterns like `faire + infinitif`

## Highest Priority

When choosing what deserves a row, prefer these first:
- direct object vs bare infinitive vs `à` vs `de`
- `que` / `si` clauses when they are core
- combination frames like `donner qqch à qqn`
- reflexive patterns that change the complement structure in a learner-relevant way

These contrasts matter more than adding extra semantic variety.

## What to Keep

- Everyday modern French
- Patterns that change grammar, meaning, or pronoun behavior
- Compact dictionary-style labels
- Short natural English glosses

## What to Avoid

- Listing every possible complement frame
- Rare or literary uses
- Awkward metalanguage in English
- Splits that are not useful for learners
- Redundant `qqn` vs `qqch` splits when one row is enough

## Preferred Style

- `parler à qqn` = speak to someone
- `parler de qqch` = talk about something
- `aimer qqn / qqch` = like; love someone / something
- `demander qqch à qqn` = ask someone for something
- `demander à qqn de + infinitif` = ask someone to do something
- `jouer à + jeu / sport` = play a game / sport
- `jouer de + instrument` = play an instrument

## Gloss Style

- Keep glosses natural and compact.
- Prefer `find; think that` over clunky explanatory English.
- Avoid placeholders like `thing`, `person`, or `someone / something adjective`.
- If a clean gloss is hard to write, the row may be too fuzzy to keep.
