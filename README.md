# Les Verbes

A French verb conjugation trainer PWA. Fast, focused daily practice with 7,000+ verbs, all major tenses, usage examples, voice dictation, and offline support.

**Live at [lesverb.es](https://www.lesverb.es)**

## Features

- Flashcard-based conjugation practice with smart frequency weighting
- 2,731 usage "nuggets" — real example sentences showing each verb in context
- All major tenses: present, passe compose, imparfait, futur, conditionnel, subjonctif, plus-que-parfait
- Learning presets (Master the Basics, Broaden Horizons, etc.)
- Voice dictation with pronunciation feedback
- Text-to-speech for questions and answers
- Full conjugation tables for every verb
- Light/dark theme, adjustable text size
- PWA — installs on phone, works offline

## Quick start

```bash
# Build (no dependencies needed beyond Python 3)
python3 build.py

# Output
open dist/index.html
```

For development, serve the source directory:
```bash
python3 -m http.server 8080
open http://localhost:8080
```

## Project structure

```
App source:
  index.html              Main app (HTML + inline scripts)
  js/script.js             App logic (flashcards, navigation, settings)
  js/practicePhrases.js    Hand-authored practice phrases
  css/style.css            Styles (light/dark, responsive)
  css/dictate-btn.css      Dictation button styles
  sw.js                    Service worker (stale-while-revalidate)
  manifest.json            PWA manifest

App data (generated):
  js/verbs.full.generated.js   7,046 verbs with full conjugation tables (compressed)
  js/verbs.full.js             Same data, uncompressed (pipeline input/output)
  verb_usages.js               2,731 usage nuggets (verb senses + example sentences)
  verb_core_patterns.json      Source-of-truth structured French core patterns
  verb_core_patterns.js        App bundle for core patterns
  reflexive_verbs_data.json    Reflexive verb conjugation data
  verb_overrides.json          Hand-curated corrections to verb data

Data pipeline scripts:
  combine_dataset_enhanced.py  Generates verbs.full.js from raw corpus data + verbecc
  fix_frequency_tiers.py       Assigns exclusive frequency buckets (top20/50/100/500/1000/rare)
  append_reflexive.py          Merges reflexive verb conjugations
  generate_verb_hints.py       Adds contextual hints via OpenAI
  apply_overrides.py           Applies manual corrections from verb_overrides.json
  compress_large_js_objects.py Gzip+base64 compression for large JS data
  generate_verb_usages.py      Generates usage nuggets via OpenAI
  generate_core_patterns.py    Generates structured French core patterns via OpenAI
  review_core_patterns.py      Syncs/checks/reviews core-pattern data
  audit_core_patterns_with_openai.py Non-destructive AI audit for core patterns
  generate_reflexive_verbs.py  Generates reflexive verb data via OpenAI

Build:
  build.py                 Bundles everything into a standalone dist/index.html
```

## Data pipeline

See [important_commands.md](important_commands.md) for full pipeline commands.

The pipeline produces two key data files:

1. **`js/verbs.full.generated.js`** — all 7,046 verb conjugation tables.
   Built from `js/verbs.full.js` via a sequence of patching scripts, then compressed.

2. **`verb_usages.js`** — 2,731 usage nuggets showing verbs in real sentences.
   Generated by `generate_verb_usages.py` using OpenAI to produce natural examples.

3. **`verb_core_patterns.json` / `verb_core_patterns.js`** — compact structured core constructions
   shown above the richer usage nuggets in the French app. The JSON file is the source of
   truth; `review_core_patterns.py sync` regenerates the JS bundle and review markdown.

Both files are tracked in git. You only need to regenerate them if you want to add verbs or improve the data.
